Time series _ Rolling means¶

Rolling means help to understand trends in time series data by smoothing out short-term fluctuations and highlighting longer-term patterns.¶
Libraries and dataset¶

The data set is randomly generated

In [23]:
import pandas as pd
import numpy as np
import warnings

warnings.filterwarnings('ignore')


# Create a sample time series DataFrame
# Using '2025-01-01' as a starting point, consistent with the current year
dates = pd.date_range(start='2022-01-01', periods=1000, freq='D')
data = {'Value': np.random.randint(80, 120, size=len(dates))}
df_ts = pd.DataFrame(data, index=dates)

Rolling means are generated for intervals of 90 and 180 days

In [24]:
# Example: Calculate a 7-90-180-day rolling mean
#rolling_mean_7day = df_ts['Value'].rolling(window=7).mean()
rolling_mean_90day = df_ts['Value'].rolling(window=90).mean()
rolling_mean_180day = df_ts['Value'].rolling(window=180).mean()

#df_ts['7-Day Rolling Mean'] = rolling_mean_7day
df_ts['90-Day Rolling Mean'] = rolling_mean_90day
df_ts['180-Day Rolling Mean'] = rolling_mean_180day
df_ts.fillna(method='bfill', inplace=True)  # Fill NaN values for plotting
